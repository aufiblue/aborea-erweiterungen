# Initiativeveränderung durch Rüstungen

Die Rüstung verändert nicht nur die Bewegungsrate, sondern auch die Initiative.

| Rüstungsart | Malus auf INI |
| - | - |
| Ungerüstet | 0 |
| Lederrüstung | 0 |
| Kettenhemd | -1 |
| Halbe Platte | -1 |
| Plattenpanzer | -2 |
| Ritterrüstung | -3 |
| Schild | 0 |
| Turmschild | -1 |
