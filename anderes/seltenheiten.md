# Seltenheiten

Folgende Tabelle mit Seltenheitswerten ist dafür da, dass man etwa einschätzen kann, wie selten Objekte in der Welt von Aborea sind.

| Seltenheitsgrad | Beschrieben mit |
| :--- | :--- |
| 1 | Sehr häufig |
| 2 | Häufig |
| 3 | Eher häufig |
| 4 | Mittel |
| 5 | Eher selten |
| 6 | Selten |
| 7 | Sehr selten |
| 8 | Äußerst selten |
| 9 | Extrem selten |
